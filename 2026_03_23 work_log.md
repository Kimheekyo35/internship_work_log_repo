1. 모달 부분에서 버튼을 통해서 `URL` 을 여는 것이었는데 `views_update` 를 사용하였더니 계속 에러가 발생함
- `views_open`: 새 모달을 처음 띄울 때 사용
- `views_update`: 이미 떠 있는 모달 내용을 바꿀 때 사용
- `views_push`: 이미 열린 모달 위에 다음 모달을 한 장 더 쌓을 때 사용

하는 것으로  `예약 취소` 버튼을 누르면 해당 모달이 뜰 수 있게 다음과 같이 작성하면 됨

```
@app.action("booking_cancel")
def click_cancel(ack, body, client):
    ack()
    user_id = body["user"]["id"]

    # 모달을 띄우려면 이렇게 해야됨
    client.views_open(
        trigger_id=body["trigger_id"],
        view=build_booking_cancel_list(get_user_future_booking(user_id))
    )
```
