### 해결 방법
1. docker recreate을 하려고 했으나 이미 돌아가고 있는 컨테이너와 충돌했음. ->
`docker rm -f meeting-room-bot-we meeting-room-bot-be meeting-room-bot-team` 로 현재 돌아가는 컨테이너 삭제함

2. yml을 수정했기 때문에 다시 build 하는 게 낫다고 생각.
  🌟 volumes는 `docker-compose.yml` 의 위치를 기준으로 잡음

3. `docker compose up -d --build --force-recreate meeting-room-bot-we meeting-room-bot-be meeting-room-bot-team` 로 컨테이너를 다시 띄움

💢 하지만 이번엔 모달을 제대로 띄우지 못한다는 에러 코드를 봄
`main.py` 에 
```
{
    "type": "section",
    "block_id": "notice_block",
    "text": {
        "type": "mrkdwn",
        "text": "* 장기 예약 문의: 위마케팅 김희교 인턴"
    }
}
```
를 추가했는데 그 전에 `"dispatch_action": True` 를 작성했었음. 이 코드는 사용자가 값을 바꾸는 순간에 앱으로 이벤트를 보내는 변수 설정임. 내가 설정한 type은 마크다운이라 에러가 발생했었음.
