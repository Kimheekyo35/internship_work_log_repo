## Notion과 Codex 연결 방법
- MCP란 ? 모델을 외부 도구와 컨텍스트에 연결하는 방식

```
# ~/.codex/config.toml
[mcp_servers.notion]
url = "https://mcp.notion.com/mcp"
```
bash나 powershell에서 codex 파일로 들어간 후 `config.toml` 을 연다.
그 다음 위 코드를 작성하고, `codex mcp login notion` 를 엔터누르면 로그인하는 흐름을 안내해 줌.
