5. shortcuts로 앱을 불러오고 싶을 떈
```
@app.shortcut("open_room_booking")
def open_booking_modal(ack, shortcut, client, logger):
    ack()

    message = shortcut.get("message", {})
    thread_ts = message.get("thread_ts") or message.get("ts")
    channel = shortcut.get("channel", {})

    open_room_booking_modal(
        client=client,
        trigger_id=shortcut.get("trigger_id"),
        logger=logger,
        source="shortcut",
        extra_metadata={
            "channel_id": channel.get("id"),
            "thread_ts": thread_ts,
        },
    )
```
이렇게 작성하여 message, thread, channel 모두에서 앱을 호출할 수 있음.
⭐ 또한, slack api에서 bot token에 이에 맞는 권한을 부여해야 함. 
https://docs.slack.dev/interactivity/implementing-shortcuts/#scope_permissions 에서 확인할 수 있음.

## Docker 복습
`dockerfile` 안에 있는 코드

```
FROM python:3.11-slim  ---> python은 베이스 이미지에 이미 포함되어 있기 때문에 FROM 으로 가져옴

WORKDIR /app           ---> app이라는 파일 안에 이미지를 저장하는 것

COPY requirements.txt .   ---> requirements에 있는 것들을 이미지에 복사함
RUN pip install --upgrade pip && pip install --no-cache-dir -r requirements.txt    ---> requirements.txt에 적힌 라이브러리들을 이미지 안에 설치

COPY . .  ---> 현재 파일에 있는 것들을 모두 /app에 복사함, .dockerignore에 있는 것들은 당연 제외함 (나는 .env를 dockerignore함)
          ---> main.py 같은 파이썬 파일들도 이미지로 복사됨

EXPOSE 9000  --> 이 컨테이너는 9000 포트를 쓸 거라고 문서처럼 표시하는 정도

CMD ["python", "main.py"]

```

.env는 이미지에 들어가지 않지만 `docker-compose.yml` 에 `env_file`로 `.env` 참고하게 함.
